Wzorzec Strategia.
Mechanizm wyboru algorytmu
Jeden z najczęściej używanych wzorców, Zgodny z zasada Open-Close dzięki czemu możemy dodać kod nie zmienajac dotychczasowego.
Wzorzec ma zastoswanie kiedy możemy rozwiazac problem na różne sposoby,
jest kilka różnych algorytmów rozwiązujących pewien problem i mamy możliwość wybory algorytmu.
1.Najpierw należy stworzyć wspólny interfejs.
2.Każdy sposób rozwiązania problemu(algorytm) trafia do klasy implementującej interfejs.
3.Utworzyć klasę kontekstu - umożliwia wybór strategii. Klasa taka zawiera pole typu interfejsowego,
seter albo konstruktor który jako parametr przyjmuje interfejs! oraz właściwą metodę.
4.Klasa klienta – tutaj tworzymy obiekt Kontekstu i wywołujemy właściwą strategie.

Zmodyfikuj poniższy kod tak aby wykorzystywał wzorzec Strategia.
public class Calculator {
public int calculate(int a, int b, String operator) {
int result = 0;
if ("add".equals(operator)) {
result = a + b;
} else if ("multiply".equals(operator)) {
result = a * b;
} else if ("divide".equals(operator)) {
result = a / b;
} else if ("subtract".equals(operator)) {
result = a - b;
}
return result;
}
}
