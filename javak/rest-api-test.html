Rest Assured służy do testowania REST API. Podbnie jak TestRestTemplate używa się do tworzenia testów automatyczych.


Rest Assured może być wykorzystany w spring boot w wersji 3.0.7


RestAssured junit integration will assume that the server is up and the app is deployed.

Powyższe oznacza, że RestAssured służy do testowania API które jest uruchomione na serwerze. Czyli jesli chcemy przetestowacć za pomoca Rest Assured API które dopiero stowrzylismy to musimy napisać odzielna aplikacje albo moduł i uruchomić testy po uruchominiu projektu. Jeżeli natomiast chcemy przetestowac controllery to mozemu uzyć MockMvc.


MockMvc:
  • służy do testów jednostokowych.
  • testuje bardziej controllery i logikę którą wykonują
  • testuje tylko „własne” API. Nie potrzebuje uruchomionego serwera.
