https://qabrio.pl/git-reset-revert/

- git reset --hard - usuwa wprowadzone zmiany. -git reset --hard HEAD~1 – usuwa ostatni komit razem ze zmianmi
-git reset --soft HEAD~1 – usuwa ostni komit bez zmian
-git reset -- hard HEAD - usuwa wszystkie komity az do wskazanego
-git commit --amend --no-edit – dodaje plik do ostatniego komita
-git revert - revert tworzy kopie comita z przed revertu. W ten sposób zapisuje historie zmian. Nie da się revertowac revertu ale można usunąć. VIM : ESC SHIFT+ALT+Srednik wq
- git rm src\main\java\pl\filewicz\Klasa1.java – usuwa zakomitowen pliki
-git rm – f – usuwa śledzone pliki
- git clean -f -usuwa nieśledzone pliki
- git clean -f - usuwa nieśledzony plik
- git reset HEAD - przestaje śledzić plik
- git rebase -i HEAD~N – scala komity w jeden VIM – fixup :wq
- git stash save ‘zmiana1’ – dodaje zmiany do poczekalni
- git stash pop stash@{0} - zastsowuje zamiany z poczekali
- git stash list – sprawdza jakie zmiany staszowe zostały zrobione
- git stash clear - czysci stasze
- git cherry-pick - wycoaga komita z jednego branacha i dodaje do drugiego
